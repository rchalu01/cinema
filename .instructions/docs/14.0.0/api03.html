Étape 2 : Projet Symfony - API avec le bundle FOS-rest

Objectifs

Mise en place du nouveau projet Symfony 4.4 API

Création du projet

  • Créer le projet Symfony "wscinema"
composer create-project symfony/skeleton:^4.4 api-v2
  • copier tout le contenu du dossier "api-v2" dans le dossier "api"
cd api-v2 
mv * ../api
mv .* ../api
  • supprimer le dossier "api-v2"
cd ..
rm -rf api-v2

Installer les bundles supplémentaires

orm

composer require orm

MakerBundle

composer require symfony/maker-bundle --dev

Annotations

composer require annotations

Serializer

composer require symfony/serializer-pack

installer FOS

Dans le conteneur Symfony API dfs-api :

composer require friendsofsymfony/rest-bundle
